# exercice_bootstrap
un exercice de bootstrap

Il s'agit d'un exercice fait sur openclassroom à l'aide de bootstrap afin d'utiliser au mieux ce framework. 
Le template devait contenir un scrollspy ainsi qu'un carousel. 

Le site contient des jumbotrons  et une opacité devait être appliqué afin de voir l'image de fond .  
Il fallait ensuite aplliquer les marges etc... pour la mise en page.
